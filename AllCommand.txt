1. php bin/magento setup:upgrade

-->The php bin/magento setup:upgrade command is used to update the Magento database schema and data, as well as apply any necessary system upgrades.


2. php bin/magento setup:di:compile

-->The above setup:di:compile command basically generates the contents of var/di folder in Magento < 2.2 and for >=2.2. According to the official Magento docs, it compiles code like Factories, Proxies, Interceptors, etc.


3. php bin/magento setup:static-content:deploy -f

-->The deployment of static view files is influenced by three Magento modes including Production mode, Default mode, and Developer mode. The static view files deployment command enables you to write the static content file when Magento 2 store is in production mode.


4. php bin/magento indexer:reindex	

-->In Magento 2 reindex is required for these 8 indexers when any changes are made:

		Category products;
		Product categories;
		Product price;
		Product entity attribute value;
		Stock;
		Catalog rule product;
		Catalog product rule;
		Catalog search.


5. bin/magento cache:flush

-->Flush Magento cache. 
-->To clean the cache in the website.
