The bin/magento setup:upgrade command is used for system upgrade or upgrade process in Magento.
When we create any other custom module or extension then this cmd is used to show our created module or extensions.

Cache Flush:
php bin/magento c:f     bin/magento cache:flush
A Magento 2 cache flush removes all stored data from the cache,A Magento 2 cache clean deletes all enabled caches. It doesn't affect other server processes that aren't part of Magento.


Di Compile:
php bin/magento s:d:c    bin/magento setup:di:compile
It will optimize the dependency injection configurations according to the area.(controller,plugin any changes)


Static Content Deploy:php bin/magento s:s:d
bin/magento setup:static-content:deploy -f 
Static content deployment in Magento 2 refers to the process of generating and integrating static files like CSS, JavaScript, and images into the Magento file system. This is particularly done when the system is set to production mode.

What is Magento 2 Reindex? Magento 2 Reindex is the process of updating the transformed data as the data changes. E.g.: when updating a product price the data should be reindexed to be correctly displayed on the frontend.

getItems() and getData(). What is the different?
->getItems() retrieves a collection of objects (e.g., products, orders, customers) from a collection.
getData() retrieves the data of a single object (e.g., product, order, customer) as an array, allowing you to access specific attributes or properties.

difference between collection and collectionFactory:
->Collection is an instance of a collection of database records (e.g., products, orders) with methods to interact with those records.
->CollectionFactory is a factory class used to instantiate Collection objects, adhering to Magento's dependency injection principles. It helps in creating instances of Collection classes without directly instantiating them using new.

For get configurable selected product id:
<script type="text/javascript">
                requirejs(['jquery','underscore'], function(jQuery,_){
                jQuery( ".product-options-wrapper div" ).on("click", function() {
                selpro();
                });
                function selpro () {
                    var selected_options = {};
                    jQuery('div.swatch-attribute').each(function(k,v){
                        var attribute_id    = jQuery(v).attr('data-attribute-id');
                        var option_selected = jQuery(v).attr('data-option-selected');
                        if(!attribute_id || !option_selected){ return;}
                        selected_options[attribute_id] = option_selected.toString();
                    });
                    var product_id_index = jQuery('[data-role=swatch-options]').data('mage-SwatchRenderer').options.jsonConfig.index;
                    var selectedSimpleProductId = '';
                    jQuery.each(product_id_index, function(product_id,attributes){
                        var productIsSelected = function(attributes, selected_options){
                            return _.isEqual(attributes, selected_options);
                        }
                        if(productIsSelected(attributes, selected_options)){
                            selectedSimpleProductId = product_id;
                        } 
                    });
                
                 window.alert(selectedSimpleProductId);
                }
            }); 
</script>
